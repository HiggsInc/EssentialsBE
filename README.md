# EssentialsBE
Based on EssentialsPE for pmmp PocketMine Servers
=========

The port version of [Bukkit Essentials](http://dev.bukkit.org/bukkit-plugins/essentials/) for [PocketMine-MP Software](http://www.pmmp.io/).

> EssentialsBE does not officially provide compatibility with _Modded PocketMine-MP_ server software.
>
> Compatibility can be found with most PocketMine based softwares, but issues might occur. (Report in Issues)
